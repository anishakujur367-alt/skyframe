#  NASA Mission Explorer

##  Description
This project shows NASA’s Astronomy Picture of the Day. Users can select a date and explore space images.

##  Features
- Date selection
- API data fetching
- Full-screen image view

##  Tech Stack
- HTML
- CSS
- JavaScript
- NASA API

##  Installation
1. Download project
2. Open index.html in browser
